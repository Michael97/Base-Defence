Click to move/shoot/gather - RMB

Interact with UI - LMB

B - Shop Menu

Esc - Pause Menu

Tutorial and tips- 

Find a good location fast (the timer starts instantly)

Place base down (500 wood, 500 stone)

Place defences around and gather resources when you are not getting attacked

Zombies drop Gold, Wood and Stone (5 for each).

Gold is currently not used.
